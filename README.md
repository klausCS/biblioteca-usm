# biblioteca-usm

Para poder trabajar en este proyecto, es necesario tener instalado **Bun** en tu sistema, ya que se utiliza para instalar las dependencias y ejecutar el proyecto.

Para compilar los archivos del proyecto, ejecutar el siguiente comando:
```bash
bun build
```
Para levantar el entorno de desarrollo y ejecutar la aplicación, utilizar el siguiente comando:
```bash
bun run dev
```
